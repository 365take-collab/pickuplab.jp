# ドメイン設定 クイックスタート（pickuplab.jp）

## ✅ 現在の状態

- ✅ Vercelプロジェクト作成完了
  - `pickuplab` または `pickuplab-jp` のどちらかを使用

---

## 🚀 次のステップ: ドメイン設定

### ステップ1: プロジェクトを選択

1. **Vercelダッシュボード** → **Projects** → **pickuplab** をクリック
   - または **pickuplab-jp** を選択（どちらでもOK）

### ステップ2: ドメインを追加

1. **Settings** タブをクリック
2. **Domains** セクションを開く
3. **Add Domain** ボタンをクリック
4. **ドメイン名を入力**: `pickuplab.jp`
5. **Add** ボタンをクリック

### ステップ3: ネームサーバー情報を確認

Vercelが以下のネームサーバー情報を表示します：

```
ネームサーバー:
- ns1.vercel-dns.com
- ns2.vercel-dns.com
```

**この情報をコピーしておきます**

### ステップ4: お名前.comでネームサーバーを設定

1. **お名前.comにログイン**: https://www.onamae.com/
2. **ドメイン一覧** → **pickuplab.jp** を選択
3. **ネームサーバー設定** → **カスタムネームサーバー**
4. 以下のネームサーバーを入力（末尾にピリオド不要）：
   - **ネームサーバー1**: `ns1.vercel-dns.com`
   - **ネームサーバー2**: `ns2.vercel-dns.com`
5. **保存** ボタンをクリック

### ステップ5: 環境変数を設定

1. **Settings** → **Environment Variables** に移動
2. 以下の環境変数を追加（**Production**環境にチェック）：

| Key | Value | Environment |
|-----|-------|-------------|
| `NEXT_PUBLIC_API_URL` | `https://api.pickuplab.jp` | Production ✅ |
| `NEXT_PUBLIC_BASE_URL` | `https://pickuplab.jp` | Production ✅ |
| `NEXTAUTH_URL` | `https://pickuplab.jp` | Production ✅ |

### ステップ6: デプロイを再実行

1. **Deployments** タブを開く
2. 最新のデプロイメントの **...** メニューをクリック
3. **Redeploy** を選択
